# 视频工具v0.0.2

- 增加过滤器对每帧图像操作，操作后的结果输出至另一显示框中
- 增加亮度与对比度调节
- 增加视频导出功能

过滤器使用单例模式，同时定义任务类XTask，通过遍历tasks依次执行任务实现任务队列，同时保证线程安全
